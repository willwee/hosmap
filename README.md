# hosmap
hospital info list

# 安装
下载完成后，在工程目录运行
npm i

# 需要注意的依赖
本项目有用到react-native-vector-icons ，需要特殊设置下，请参考react-native-vector-icons的安装。其余的没有需要注意的点。
